A plugin to upload GSA file from lab.

== Description ==

A simple file uploader for GSA files. After uploading the user details are matched with the respective barcode from gsa.

== Usage ==

To view the upload tool, place the shortcode `[gsa-upload]` in a new full-width page, through editor.
You can also use `<?php do_action('gsa-upload'); ?>` if you are planning to hardcode the page.
